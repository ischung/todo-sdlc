# 사용자 가이드 — 첫 5분

이 문서는 todo-sdlc 를 처음 만난 사용자가 5분 안에 핵심 흐름을 익히는 것을 목표로 합니다.
스크린샷 자리에는 캡쳐 후 `docs/img/` 에 PNG 로 저장하면 됩니다 (현 단계는 placeholder).

> 어디서 쓸 수 있나요?
>
> - 스테이징(상시 최신): <https://ischung.github.io/todo-sdlc/staging/>
> - 프로덕션(승인된 릴리스): <https://ischung.github.io/todo-sdlc/>
> - 로컬: `npm install && npm run dev` → <http://localhost:5173>

---

## 0. 화면 구성

![calendar overview](./img/00-overview.png)

- 상단: 현재 달 + 이전 달 / **오늘** / 다음 달 버튼
- 중앙: 7×6 달력 그리드 — 오늘은 **굵은 파란색 숫자**로 강조
- 셀 우하단: 미완료 todo 가 1 개 이상이면 **파란 동그라미 배지** (10 개 이상이면 `9+`)

## 1. 할 일 추가하기 (F2)

![add todo](./img/01-add.png)

1. 원하는 날짜 셀을 **클릭**(또는 키보드로 포커스 후 `Enter`).
2. 하단 시트(모바일) 또는 가운데 카드(데스크탑)가 열립니다.
3. 입력란에 내용을 적고 **Enter**. 빈 입력은 거부되고 입력란이 살짝 흔들립니다.
4. 입력란은 비워지고 **포커스가 유지**되어 연속 입력이 가능합니다.
5. 100 자 초과 입력은 자동으로 잘립니다.

## 2. 완료 표시 (F3) · 미완료 배지 (F5)

![toggle done](./img/02-toggle.png)

- 좌측 체크박스를 클릭(또는 `Space`)하면 완료/미완료가 토글됩니다.
- 완료된 항목은 **취소선 + 회색 톤** 으로 바뀌고, 셀의 미완료 배지가 1 줄어듭니다.
- 모든 항목이 완료되면 셀의 배지는 사라집니다.

## 3. 삭제 + 5초 되돌리기 (F4)

![delete and undo](./img/03-undo.png)

1. 항목 우측의 `✕` 버튼으로 삭제.
2. 화면 하단에 **5초간** "되돌리기" 토스트가 떠 있습니다.
3. **되돌리기** 를 누르면 같은 위치로 복원되고 영속 저장도 되돌아갑니다.
4. 5초가 지나면 토스트는 사라지고 삭제가 확정됩니다.

## 4. 다른 달로 이동 (F1 / F8)

![month navigation](./img/04-nav.png)

- `←` / `→` 버튼으로 이전/다음 달.
- **오늘** 버튼은 현재 월일 때 비활성화되며, 다른 달에 있을 때 누르면 즉시 현재 월 + 오늘 강조 상태로 돌아옵니다.

## 5. 새로고침해도 사라지지 않아요 (F6)

- 데이터는 브라우저의 `localStorage` 키 `todo-sdlc/v1` 에 저장됩니다.
- 새로고침/재방문에도 그대로 유지됩니다.
- 같은 브라우저의 다른 탭에서 추가/삭제하면 이 탭도 자동 갱신됩니다 (`storage` 이벤트).

## 6. 시크릿 모드 / 저장소 비활성

저장소가 비활성화된 환경(시크릿 모드 등)에서는 상단에 안내 배너가 나타납니다. 이 모드에서는 데이터가 저장되지 않으니 일반 창으로 다시 열어주세요.

## 7. 키보드만으로 사용하기 (a11y)

| 키 | 동작 |
| :---- | :---- |
| `Tab` | 다음 인터랙티브 요소로 이동 |
| `Enter` / `Space` | 셀 선택, 버튼 활성, 체크박스 토글 |
| `Esc` | 패널 닫기 |

모든 인터랙티브 요소에는 키보드 포커스 시 파란 링이 표시됩니다.

## 8. 막혔을 때

- 추가가 안 돼요 → 시크릿 모드 안내 배너가 보이는지 확인하세요.
- 저장 실패 토스트가 떴어요 → 디스크/저장소가 가득 찼을 가능성. 오래된 항목을 정리하거나 다른 브라우저로 시도해주세요.
- 더 깊은 문제는 [`docs/RUNBOOK.md`](./RUNBOOK.md) 의 "자주 나는 이슈" 절을 참고.
